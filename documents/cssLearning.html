CSS Cheat Sheet (click to copy)

  • * → all elements
  • div → element selector
  • .child → class selector
  • #main-title → ID selector
  • [disabled] → attribute selector
  • .parent > .child → direct descendant
  • .child + .sibling → adjacent sibling
  • .child ~ .sibling → general sibling
  • [href$='.doc'] → ends with
  • [href^='/index'] → begins with
  • [class*='box'] → contains substring
  • :first-child → first child
  • :last-child → last child
  • :nth-child(2) → second child
  • :only-child → only child
  • :checked → checked inputs
  • :disabled → disabled inputs
  • :empty → empty elements
