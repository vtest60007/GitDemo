XPath Cheat Sheet (click to copy)

  • //h1 → all h1 elements
  • //ul/li[1] → first li in each ul
  • //ul/li[last()] → last li in each ul
  • //a[contains(@href,'://')] → external links
  • //li[@id='item2']/ancestor::section → ancestor section of item2
  • //strong[@id='deep']/parent::span → parent span of strong
  • //li[@id='item1']/following-sibling::li → siblings after item1
  • //comment() → comment nodes
  • //processing-instruction() → processing instructions
  • //div/text() → text nodes inside div
  • //ex:item → namespaced elements
  • normalize-space(//p[@id='whitespace']) → trims whitespace (string result)
  • substring(//p[@id='substring']/text(),1,5) → first 5 chars (string result)
  • translate(//p[@id='translate']/text(),'ABC','XYZ') → replace chars (string result)
